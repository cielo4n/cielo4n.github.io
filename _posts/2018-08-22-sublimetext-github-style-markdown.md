## 2. 서브라임 텍스트에서 깃허브 스타일 마크다운 보기
서브라임 텍스트에서 단축키 `Ctrl+Shift+P` 누르고 아래 설치
```
Package Control:Install Package
MarkdownEditing
```
서브라임 텍스트 설정 변경  
`View > Syntax > Open all with current extension as > MarkdownEditing > Markdown GFM`
